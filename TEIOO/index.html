TEI and Open Office

Home

This package provides import and export XSLT stylesheets, a template file, to allow Open Office (version 1.1 and later) to open TEI XML documents, and save OO documents as TEI XML. The XSLT stylesheets effectively provide a mapping between the TEI DTD and the OpenOffice schema.

You can download the files as a zip archive. To install them, start Open Office and and follow the Tools / XML Filter Settings menu. Make a New filter and specify the location of the .xsl files and the .stw file in the Transformations panel. You'll also need to specify your DTD location.
