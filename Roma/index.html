Roma: generating customizations for the TEI

 

These pages will help you design your own TEI customization, from which you can generate DTD, RELAX NG, W3C Schema and Schematron schemas, or produce documentation.

Create a new or upload existing customization

  • Build up: create a new customization by adding elements and modules to the smallest recommended schema
  • Reduce: create a new customization by removing elements and modules from the largest possible schema
  • Create customization from template
  • Create customization from Open existing customization A number of community-maintained customizations are listed on the TEI wiki; you can download their ODD files, and upload them to Roma here.
